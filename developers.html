Fairfax FLAGS (Foreign Language Advocacy for Grade Schools) is a group that advocates for foreign language education in elementary schools in Fairfax county Virginia. The site contains maps, data and applications to support that advocacy.

Site adapts the "Modern Business" bootstrap theme from Start Bootstrap

It uses harp.js for simple html templating.

The links to the right are for review before publishing to the main Fairfax FLAGS site
